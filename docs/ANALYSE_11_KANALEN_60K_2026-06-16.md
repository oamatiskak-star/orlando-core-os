## Conclusie

- **Satisfying/LEGO-kanalen zijn het best faceless-maakbaar, maar het slechtste €-pad:** RPM €0,3–2 → €60k vereist 30–200M views/maand én vrijwel geen affiliate. Ondanks 992 video's al 0 tractie.
- **Het enige kwadrant dat tegelijk hoog-omzet ÉN faceless-maakbaar ÉN bewezen-intel is: NL-finance long-form data-explainer.** €60k vereist "maar" ~4–7M views/maand (12× minder dan satisfying) plus een groot affiliate-plafond (brokers/cursussen).
- **Markt-keuze:** NL = bewezen intel + taalmoat + minder concurrentie (kleinere markt). EN/US-finance = veel groter + hogere RPM (€15–40) maar brute concurrentie voor een faceless kanaal.
