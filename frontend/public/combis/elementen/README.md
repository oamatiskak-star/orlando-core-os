# Element-foto's (tussenlaag)

Foto's per **element/regel** binnen een combi — de tussenlaag die opengaat
wanneer je een combi opent of een regel uitklapt (raamkozijnen, deuren,
platte/hellende daken, enz.). Elke regel toont automatisch zijn foto zodra
het bestand hier staat; anders valt de regel terug op het combi-icoon.

## Vereisten
- **Formaat:** JPG, liggend, ca. 400×300 px.
- **Rechten:** uitsluitend royalty-vrije / eigen foto's. Geen beeld uit
  2Jours / Raabcalc of betaalde bibliotheken kopiëren.
- **Naamgeving:** kleine letters, spaties/leestekens → koppelteken
  (de `slug()`-regel uit `page.tsx`). Hieronder staan alle exacte namen.
